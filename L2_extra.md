# 遊戲: 捉點

.blockquote[
.title[Coding Plan]

```
創建變量`dot`為遊戲⻆色 (`game.createSprite()`)
讓它在畫面中間一行左右移動
當它在畫面正中間時按A 鍵捉住它𠌊來得分
玩遊戲15秒

挑戰:
要𤫟玩家取得10分來結束遊戲
```

]

.footnote[
[Snap the Dot - Microsoft MakeCode](https://makecode.microbit.org/projects/snap-the-dot)  
[Program: snap-the-dot](https://makecode.microbit.org/_eJrbk2EbLaDX)
]
